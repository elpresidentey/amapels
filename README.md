# AMAPELS NG — Digital Luxury Fashion House

A premium Nigerian fashion e-commerce platform inspired by Magda Butrym.

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack

- **Frontend**: Next.js 14 + TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Backend (coming soon)**: Supabase
- **Payments (coming soon)**: Paystack / Flutterwave

## Features

- ✅ Luxury homepage with bento grid layout
- ✅ Collections page
- ✅ Shop with filters
- ✅ Product detail pages
- ✅ Brand story page
- ✅ Responsive design
- ✅ Smooth animations
